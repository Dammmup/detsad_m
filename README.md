# Детсад Управление - Мобильное приложение

Мобильное приложение для управления детским садом, разработанное с использованием Flutter. Предоставляет удобный интерфейс для родителей, воспитателей и администраторов на мобильных устройствах.

## Архитектура

Мобильное приложение построено на Flutter с использованием архитектурного подхода MVVM (Model-View-ViewModel). Приложение разделено на слои: представление, бизнес-логика и работа с данными.

### Структура проекта:
- `lib/main.dart` - точка входа в приложение
- `lib/screens/` - экраны приложения
- `lib/widgets/` - переиспользуемые виджеты
- `lib/models/` - модели данных
- `lib/services/` - сервисы для работы с API и бизнес-логикой
- `lib/providers/` - провайдеры состояний (если используется Provider)
- `lib/core/` - основные компоненты приложения (константы, ошибки, базовые классы)
- `lib/utils/` - утилиты и вспомогательные функции

### Технологии:
- Flutter (Dart)
- Provider или Bloc для управления состоянием
- HTTP клиент для взаимодействия с API
- Local notifications для уведомлений
- Firebase для пуш-уведомлений
- SQLite для локального хранения данных (при необходимости)

## Установка и запуск

1. Убедитесь, что у вас установлен Flutter SDK
2. Клонируйте репозиторий
3. Перейдите в директорию проекта
4. Выполните команду: `flutter pub get`
5. Для запуска на устройстве: `flutter run`

Для сборки под конкретную платформу используйте:
- Android: `flutter build apk`
- iOS: `flutter build ios`
